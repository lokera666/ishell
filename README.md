# ishell

ishell 是一个跨平台的shell命令行、远程文件管理工具

可以运行在windows Linux（Ubuntu、统信UOS、Deepin） Mac 平台

## 编译

### WIN64

  ```
  MSVC+libssh2+openssl+QT5.12.10+Cmake
  ```
  libssh2和openssl 库在win分支Libs目录下

### Unix/Linux

  ```
  GCC+libssh2+openssl+QT5.12.10+Cmake(Qtcreator)
  ```
